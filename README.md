# P-Reinforce: Autonomous Knowledge Gardener

P-Reinforce is an AI-driven knowledge automation agent that merges Andre Karpathy's LLM-Wiki architecture with Reinforcement Learning principles.

## 🚀 How it Works
1. **Ingest**: Place any raw notes, ideas, or fragments into `00_Raw/YYYY-MM-DD/`.
2. **Process**: The P-Reinforce Architect (AI) monitors these inputs.
3. **Categorize**: It autonomously places files into `10_Wiki/` based on semantic similarity.
4. **Connect**: Documents are linked bi-directionally, updating `20_Meta/Graph.json`.
5. **Sync**: All changes are automatically committed and pushed to GitHub.

## 📂 Structure
- `00_Raw/`: Source of truth for raw inputs.
- `10_Wiki/`: Structured knowledge base.
- `20_Meta/`: System intelligence (Policy, Graph, Index).
